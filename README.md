# AC Universal Remote Control — Privacy Policy

This repository hosts the public privacy policy for the **AC Universal Remote Control** Android app (`com.ac.air.conditioner.acremote`).

**Live URL:** https://arsh1219.github.io/acremote-privacy/

Use that URL in:
- Google Play Console → Store listing → Privacy policy
- Google Play Console → App content → Data safety
- The app's `lib/core/constants/app_info.dart` (`privacyPolicyUrl`)

The page is a single self-contained `index.html` (no build step). Edit it and commit to update the live policy.
